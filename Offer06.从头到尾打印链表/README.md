# [剑指Offer 06. 从头到尾打印链表](https://leetcode-cn.com/problems/cong-wei-dao-tou-da-yin-lian-biao-lcof/)

## 题目内容

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

- 示例1:
  - 输入：\
    head = [1,3,2]
  - 输出：\
    [2,3,1]

- 限制：
  - 0 <= 链表长度 <= 10000

## 解题思路

### 递归

1. 先遍历至链表末端，回溯时依次把节点值加入列表

### 辅助栈

1. 定义两个栈
2. 入栈：遍历整个链表，将各个节点push入栈
3. 出栈：将各个节点值pop出栈，然后依次加入数组
